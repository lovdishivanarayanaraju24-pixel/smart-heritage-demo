# User Manual: Smart Heritage Demo

Welcome to the Smart Heritage Demo! This guide will help you understand how to use the application.

## Getting Started
1. Clone the repository.
2. Install dependencies with `npm install`.
3. Set up environment variables by copying `.env.example` to `.env.local` (or `.env`).
4. Run the development server with `npm run dev`.

## Features
- **Explore**: Browse through heritage sites.
- **Image Recognizer**: Upload or capture an image to identify historical monuments.
- **Chatbot**: Interact with our AI assistant to learn more about the heritage sites.

## Troubleshooting
If you encounter any issues, please open an issue on the GitHub repository.
