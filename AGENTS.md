# Agent Architecture

This document describes the AI agents and autonomous systems used in the Smart Heritage Demo.

## Overview
The project utilizes agents for various automated tasks, including:
- **Chatbot Agent**: Handles user queries related to heritage sites using an LLM.
- **Vision Agent**: Processes images to identify and tag historical landmarks.

## Configuration
Agent behaviors and prompts can be configured via environment variables. Ensure the necessary API keys (like OpenAI) are provided.
